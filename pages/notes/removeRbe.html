# 去掉ios的橡皮筋效果与安卓机的炫光效果

  • 安装inobounce.js
    • npm inobounce install --save
  • 使用
    • 页面引入 import inobounce from "inobounce"
    • 页面加载时调用 inobounce.enable()
      mounted () {
    inobounce.enable()
  }
    • 页面退出时调用 inobounce.disable()
      beforeDestroy () {
    inobounce.disable()
  }
