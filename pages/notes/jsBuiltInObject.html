# JS基本内置对象

# 值属性

# Infinity

  • Infinity是全局对象global object的一个属性,表示正无穷大，为只读属性。
  console.log(Infinity) // Infinity
  console.log(Infinity+1) // Infinity
  console.log(1/Infinity) // 0

# NaN

  • 表示不是一个数字，为Not A Number的缩写
    • NaN 属性的初始值就是 NaN，和 Number.NaN 的值一样。在现代浏览器中（ES5中）， NaN 属性是一个不可配置（non-configurable），不可写（non-writable）的属性。
  • 判断一个值是否是NaN
    • 等号运算符==和===不能判断一个值是否是NaN。必须使用Number.isNaN()或isNaN()函数。在执行自比较之中：NaN不等于它自己。
      NaN === NaN // false
  Number.NaN === NaN // false
  isNaN(NaN) // true
  isNaN(Number.NaN) // true

# undefined

  • 全局属性undefined表示原始值undefined，为JavaScript的原始数据类型，不能被配置，不能被重写
    • 一个没有被复制的变量的类型是undefined。如果方法或者语句中操作的变量没有被赋值，则会返回undefined。
      function test(a){
    console.log(a) // undefined
    return a
  }
  test () //undefind
  • 严格相等和undefined
