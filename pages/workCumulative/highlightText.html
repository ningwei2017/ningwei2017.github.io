# 文字高亮组件

# 实现效果

  • 项目当中遇到后台返回的文字需要把一些特殊词语高亮如:’温馨提示，如需开具发票，请在购买后30天内提交申请‘,如图一。

当然，如果不是后台返回的数据我们直接span标签包裹，然后给class名跟样式就好了，但是后台返回的是一整串字符串，所以我们需要通过自己编写组件来实现对应的方法。

# 实现思路

  • 我们只要获取到对应的展示标签下的内容，然后通过replace()方法把需要高亮的文字替换成利用span包裹加上class样式的就好了，如：
  <template>
    <div>
      <p class="highlight">{{message}}</p>
    </div>
  </template>
  <script>
  export default {
    name:'****',
    data () {
      return {}
    },
    mounted(){
      this.$nextTick(function () {
        const highlightBox = document.getElementsByClassName('highlight')[0]
        let highlightText = highlightBox.innerText
        const highlightArray = ['***','***']
        highlightArray.forEach(item=>{
          const rText = new RegExp(item,'g')
          const sText = `<span style="color:red;">${item}</span>`
          highlightText = highlightText.replace(rText,sText)
        })
        highlightBox.innerHTML = highlightText
      })
    }
  }
  </script>
  • 这样写虽然可以实现，但是如果项目大，用到的地方比较多，每次都需要复制代码，增加的项目的代码量与体积。为什么我们不写一个对应的组件来实现这个方法呢。

# 封装组件

  1. 创建highlightText.vue文件，编写一下内容
  <script>
  export default {
    name: 'HighlightText',
    props: {
      text: {
        type: String,
        default: ''
      },
      hKey: {
        type: Array,
        default: () => {
          return []
        }
      },
      color: {
        type: String,
        default: ''
      }
    },
    render: function(createdElement) {
      let domText = this.text
      const keyArray = this.hKey
      keyArray.forEach(item => {
        const rText = new RegExp(item, 'g')
        const sText = `<span style="color:${this.color}">${item}</span>`
        domText = domText.replace(rText, sText)
      })
      return createdElement('span', {
        domProps: {
          innerHTML: domText
        }
      })
    }
  }
  </script>
  1. 对应页面使用引入
  <template>
    <div>
      <highlightText :text="message" :hKey="hKey"></highlightText>
    </div>
  </template>
  <script>
  import highlightText from '@/components/highlightText'
  export default {
    name:'****',
    components:{highlightText},
    data () {
      return {
        message:'',
        hKey:['100','200']
      }
    }
  }
  </script>
